# Snat's Simple FFXIV's Overlay

A lightweight web-based overlay for Final Fantasy XIV. The idea of the overlay is for those who want a full-screen overlay to use within a web browser. Designed to be used as a local browser such as  to display on-screen information using HTML, CSS and JavaScript.

## Features

- Lightweight and self-contained.
- Suitable for use as a local Browser Source (OBS).
- Simple customisation via CSS and JavaScript.

## Requirements

- An overlay host or browser capable of loading local HTML (e.g. OBS Browser Source).
- A modern web browser for testing (Chrome, Edge, Firefox).
- It is **NOT** designed to be overlayed over Final Fantasy XIV.

## Quick start

1. Point your overlay host at `index.html` or set a Browser Source to the project folder.
2. Open `index.html` in a browser for quick testing.
3. Change layout and colours in `overlay.css`.
4. Modify interactions and behaviour in `overlay.js`.

## Files

- `index.html` — Main overlay markup.
- `overlay.css` — Styles and layout.
- `overlay.js` — Overlay behaviour and logic.

## Customisation

Edit `overlay.css` to change layout, fonts and colours. Edit `overlay.js` to adjust functionality and initialise any data you wish to display.
